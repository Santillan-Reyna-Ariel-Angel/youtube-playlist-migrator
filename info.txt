# Guía completa: Migrar playlists de YouTube con la API oficial

## 1. Crear credenciales en Google Cloud
1. Ve a [Google Cloud Console](https://console.cloud.google.com/).
2. Activa la API **YouTube Data API v3** en tu proyecto.
3. Entra a **APIs & Services → Credentials**.
4. Crea un **OAuth 2.0 Client ID**:
   - Tipo: **Desktop App**.
   - Obtendrás dos valores:
     - **Client ID**
     - **Client Secret**
5. Guarda ambos en un lugar seguro. El **Client Secret** es la clave privada asociada a tu Client ID.  
   - Si lo perdiste, puedes regenerarlo en la misma sección.

---

## 2. Instalar dependencias
En tu PC, instala Node.js y la librería oficial de Google:

```bash
npm install googleapis
```

---

## 3. Configurar OAuth
El flujo OAuth se hace en dos fases:
- **Cuenta origen** → para leer la playlist.
- **Cuenta destino** → para crear la nueva playlist y agregar videos.

Ejemplo de configuración:

```js
const { google } = require("googleapis");

const oauth2Client = new google.auth.OAuth2(
  "TU_CLIENT_ID",
  "TU_CLIENT_SECRET",
  "http://localhost:3000/oauth2callback"
);

const youtube = google.youtube({ version: "v3", auth: oauth2Client });
```

---

## 4. Generar URL de autorización
Antes de usar el script, necesitas autenticarte en tu cuenta de YouTube:

```js
const scopes = ["https://www.googleapis.com/auth/youtube"];
const authUrl = oauth2Client.generateAuthUrl({
  access_type: "offline",
  scope: scopes,
});
console.log("Visita esta URL para autenticar:", authUrl);
```

1. Copia la URL en tu navegador.
2. Inicia sesión en tu cuenta de YouTube.
3. Copia el **código de autorización** que te da Google.

---

## 5. Intercambiar el código por tokens
```js
oauth2Client.getToken("CODIGO_DE_AUTORIZACION", (err, tokens) => {
  if (err) return console.error("Error al obtener tokens:", err);
  oauth2Client.setCredentials(tokens);
  console.log("Autenticación completada.");
});
```

⚠️ Haz este proceso primero con la **cuenta origen** y luego con la **cuenta destino**.

---

## 6. Obtener videos de la playlist origen
```js
async function getPlaylistItems(playlistId) {
  const res = await youtube.playlistItems.list({
    part: "snippet",
    playlistId,
    maxResults: 50,
  });
  return res.data.items.map(item => item.snippet.resourceId.videoId);
}
```

---

## 7. Crear playlist destino
```js
async function createPlaylist(title) {
  const res = await youtube.playlists.insert({
    part: "snippet,status",
    requestBody: {
      snippet: { title },
      status: { privacyStatus: "private" },
    },
  });
  return res.data.id;
}
```

---

## 8. Insertar videos en la playlist destino
```js
async function addVideoToPlaylist(playlistId, videoId) {
  await youtube.playlistItems.insert({
    part: "snippet",
    requestBody: {
      snippet: {
        playlistId,
        resourceId: { kind: "youtube#video", videoId },
      },
    },
  });
}
```

---

## 9. Flujo completo
```js
(async () => {
  const origen = "ID_DE_TU_PLAYLIST_ORIGEN";
  const videos = await getPlaylistItems(origen);

  const destino = await createPlaylist("Mi Playlist Migrada");

  for (let v of videos) {
    await addVideoToPlaylist(destino, v);
    console.log(`Agregado: ${v}`);
  }
})();
```

---

## 10. Notas importantes
- Necesitas autenticarte **dos veces**: primero en la cuenta origen, luego en la cuenta destino.
- El **Client Secret** se obtiene en la misma sección donde generaste tu Client ID.
- Si tu playlist es privada, OAuth es obligatorio.
- La **API key sola no basta** para escribir en tu cuenta.
- Puedes migrar cientos de videos (ej. tus 400) sin problema.

---

## ✅ Conclusión
Este método es:
- **Fiable** → usa la API oficial, no depende del DOM.
- **Escalable** → funciona con cientos de videos.
- **Seguro** → controlas acceso con OAuth y tus credenciales.
